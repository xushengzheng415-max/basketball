# 赛小蜂篮球 1.0 视觉原型与开发说明

> 说明：本组原型为 PNG 图片格式，面向产品沟通、软著截图、开发对齐使用。

## 二、视觉风格

- 主色：橙色 `#ff4d18`，用于主按钮、比分、重点行动。
- 辅色：黄色 `#ffd956`，用于 MC 状态、比赛节次、强调标签。
- 深色：黑蓝 `#07131d`，用于导航、计分盘背景、后台侧边栏。
- 背景：浅灰蓝 `#f3f6fa`，保持页面干净。
- 卡片：白色卡片，8-10px 圆角，不做过度装饰。
- 字体：优先使用微信环境默认中文字体，设计稿按微软雅黑/PingFang SC 处理。

## 三、核心组件

### 1. 顶部导航

- 适用页面：登录、首页、快速比赛、我的、音效设置。
- 结构：
  - 深色背景。
  - 中间标题。
  - 左侧返回箭头。
  - 微信右上角胶囊由系统提供，不建议自绘。

### 2. 主按钮

- 橙色按钮：核心动作，例如开始比赛、保存设置、结束比赛。
- 黄色按钮：强调状态动作，例如开始计时、MC 当前播放项。
- 深色按钮：次级动作，例如退出登录、音效选项。
- 白色描边按钮：可切换项，例如正计时、固定播放。

### 3. 信息卡片

- 首页最近赛果、赛事卡、球员卡、我的账号卡均使用卡片。
- 卡片内部保持信息少而清楚。
- 不做卡片套卡片。

### 4. 计分队伍卡

字段：
- 队名
- 当前比分
- 犯规次数
- 暂停次数
- `+1`
- `+2`
- `长按+3`

交互：
- 点 `+1`，对应队伍加 1 分。
- 点 `+2`，对应队伍加 2 分。
- 长按 `长按+3`，播放 3 分音效并显示长按进度，进度完成才加 3 分。
- 犯规和暂停支持加减。

### 5. MC 抽屉

位置：
- 左侧隐藏式抽屉。

入口：
- 计分盘左侧 `MC` 竖向标签。

内容：
- 欢呼
- 投篮未进
- 2分有效
- 休息音乐
- 进攻音乐
- 防守音乐

规则：
- 所有音频统一单路播放，不允许叠加。
- 进攻和防守是互斥循环播放。
- 当前播放按钮高亮为黄色。
- 再点当前播放按钮则暂停。

### 6. 外设快捷键

1.0 展示为独立能力，但开发可以和 MC 设置共用数据结构。

字段：
- 6 个按键位。
- 每个键位绑定一个音效类型。
- 支持监听蓝牙小键盘键值。

默认：
- 1：欢呼
- 2：2分
- 3：3分
- 4：休息
- 5：进攻
- 6：防守

### 7. 音效设置项

音效类型：
- 2分音效
- 3分音效
- 休息音乐
- 防守音乐
- 进攻音乐
- 蜂鸣器
- 比赛开始音效
- 氛围音效
- 播报语音包

播放策略：
- 固定：使用当前选中的一首。
- 随机：从该分类音乐库中随机取一首。

注意：
- `2分音效`、`3分音效`、`蜂鸣器`、`比赛开始音效` 默认固定。
- `休息音乐`、`防守音乐`、`进攻音乐`、`氛围音效` 默认随机。

## 四、关键交互流程

### 1. 游客体验

登录页点击 `游客体验`：

1. 进入快速比赛设置。
2. 不进入首页。
3. 体验基础计分功能。
4. MC 功能提示分享解锁。

### 2. 微信登录

登录页点击 `微信一键登录`：

1. 拉起微信授权。
2. 获取昵称、头像、openid。
3. 手机号授权成功后展示在我的页面。
4. 登录成功进入首页。

### 3. 分享解锁 MC

用户点击 MC 锁定提示：

1. 弹出分享解锁说明。
2. 文案：分享到朋友圈，解锁 1 天 MC 音效会员。
3. 用户完成分享后，云端写入 1 天权益。
4. 计分盘显示 `MC 音效已解锁`。

### 4. 直接付费

入口不放在首页显眼位置。

位置：
- 分享弹窗的次级按钮。
- 我的页面会员区域。

套餐：
- 9.9 / 天
- 19.9 / 月
- 99 / 永久

### 5. AI 播报包

不作为首页主卖点。

触发：
- 播报次数快用完时提醒购买。

语音包名称：
- 赛小智：男声
- 赛小瑜：女声
- 赛小萌：童声
