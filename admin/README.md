# 赛小蜂篮球 PC 后台原型

这是一个静态 PC 后台原型，用于先确认后台需要看的数据结构和页面布局。

## 打开方式

直接用浏览器打开：

`admin/index.html`

或者在项目根目录启动一个静态服务：

```powershell
npm run admin:serve
```

然后访问：

`http://localhost:5174`

## 当前能力

- 查看小程序用户资料：昵称、微信授权标识、授权手机号、登录来源、更新时间。
- 查看用户反馈：类型、内容、联系方式、关联用户、提交时间。
- 查看基础数据概览：用户数、反馈数、赛事数、场次数、球员数。
- 支持导入 JSON 数据文件，格式可参考 `admin/sample-data.json`。

## 微信授权说明

手机号应通过微信手机号授权流程获取并由服务端解密保存。真实微信号通常不会通过小程序 API 直接开放，后台更适合存储 `openid` / `unionid` 这类微信授权标识。

## 后续接真实数据

当前小程序数据还存在用户手机本地存储里，PC 后台不能直接读取每台手机里的本地数据。正式上线时建议接微信云开发或自建服务端，把 `userProfile`、`feedbackList`、赛事、场次、球员数据提交到云端数据库，后台再读取云端数据。
