# 赛小蜂篮球

赛小蜂篮球小程序前端项目，第一阶段聚焦赛事首页、比赛计分台、赛事运营、球队和技术统计。

## 技术栈

- Taro
- React
- TypeScript

## 本地开发

```bash
npm install
npm run dev:weapp
```

然后用微信开发者工具打开当前目录，开发构建产物在 `dist/`。

## 原型范围

- 首页：今日比赛、运营指标、快速计分、赛事入口、运营提醒
- 计分台：比分、节次、计时、犯规、暂停、阵容、技术统计、事件流、撤销和比赛控制
- 赛事：赛程、积分榜、场地和运营异常
- 球队：阵容、赛程、球员数据
- 统计：球队对比、球员榜单、关键技术统计
